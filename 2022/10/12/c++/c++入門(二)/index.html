c++筆記(二)-運算式與運算子

發表於2022-10-12|更新於2022-10-18
|閱讀量:

算數運算子

算數運算子 意義 備註
+ 加
- 減
* 乘
/ 除
% 取餘數
++ 遞增 i++ 即 i=i+1
-- 遞減 i-- 即 i=i-1

浮點數的類型(float, double)無法取餘數!!!


關係運算子

關係運算子 意義 備註
== 等於 = 是指派，== 是判斷是否相等
!= 不等於
> 大於
>= 大於等於
< 小於
<= 小於等於 i++ 即 i=i+1

a < b < c 要寫成 a < b && b < c !!!

邏輯運算子

邏輯運算子 意義 備註
&& and
|| or
! not

總結
