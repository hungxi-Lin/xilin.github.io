c++筆記(一)-基本使用

發表於2022-10-12|更新於2022-10-17
|閱讀量:

基本輸出與輸入(visual.studio)

  • Ctrl+滾輪(放大、縮小)

  • F5 (Build and run)

  • Ctrl+a(全選)，Ctrl+c(複製)，Ctrl+v(貼上)，Ctrl+s(存檔)

  • F11(逐步執行)

  • //(註解)

第一個程式

解釋

#include <iostream>

  • #include 將尖括號<>内指定的文件包含到本程序中，可以想像為**餐廳菜單中的不同分類，像是定食、甜點、飲品…**，若我們想要選擇甜點，則必須到甜點這頁來尋找，所以就變成#include<甜點>。

  • <iostream> 被稱為標頭檔，依照自己的需求來添加 ( 後面會細說 )，常見的有 <iostream>、<cmath>、<iomanip>….。

using mamespace std;

  • 目前可以先理解為存儲預設標頭檔的資料庫，詳情可見 詳細資料

C語言程式是由許多函式 ( function ) 所組成，一定包含一個主函式 main()。

  • 程式是從 main() 開始執行。

int main()

  • 傳回值的型態，main()做完後會傳回一個整數 ( 為一種變數型態，下方會提到 )

return 0;

  • 函式結束並傳回一個值0

cout

  • 標準輸出到 monitor or cmd
  • cout << 字串常數；
  • 字串常數前後用 ” 括起來

endl

  • 這裡表示換行，通常搭配 cout 使用
  • cout << "字串常數" << endl

變數型態

常用型態 意義 範圍
int 整數 -2147483648 ~ 2147483647
long long int 長整數 -2^63 ~ 2^63-1
float 單精度浮點數(小數) ±1.5x10^−45 ~ ±3.4x10^38
double 雙精度浮點數(小數) ±5.0×10^−324 ~ ±1.7×10^308
char 字元 ‘A’
string 字串 ‘Hello’
bool 布林 true、false

變數宣告

總結

這一次帶大家基礎入門了c++中最基礎卻也至關重要的一環基本輸出輸入、基礎變數與宣告，以下將會列舉幾個新手在這章節易犯的錯誤。

  • cin >> a,b; error
    cin >> a >> b; correct

  • 忘記加上 ;

  • int math=999999999; error
    long long int math=999999999; correct
